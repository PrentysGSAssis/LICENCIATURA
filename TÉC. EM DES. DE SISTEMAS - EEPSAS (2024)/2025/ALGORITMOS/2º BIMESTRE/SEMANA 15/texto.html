Explorando os Tipos de Dados

Prezados futuros Programadores, na semana anterior, introduzimos o conceito fundamental de variáveis como "caixas" para guardar informações que podem mudar. Vimos os tipos inteiro e cadeia. Hoje, expandiremos nosso conhecimento explorando outros tipos de dados básicos que o Portugol Studio oferece: real e caracter. Compreender esses tipos é essencial, pois cada um é adequado para armazenar um tipo diferente de informação, otimizando o uso da memória e permitindo operações específicas. Lembrem-se, estamos construindo a lógica de Programadores, instruindo o computador a realizar tarefas.

Recapitulando e Expandindo:

  • inteiro: Armazena números inteiros. 
    
programa {
    inteiro numero_original, resultado_soma

    funcao inicio() {
        escreva("Digite um número inteiro: ")
        leia(numero_original)
        resultado_soma = numero_original + 10 // Soma 10 ao número lido
        escreva(numero_original, " + 10 = ", resultado_soma, ".\n")
    }
}
  • cadeia: Armazena sequências de caracteres (texto), entre aspas duplas. 
    
programa {
    cadeia nome_usuario

    funcao inicio() {
        escreva("Qual é o seu nome? ")
        leia(nome_usuario)
        escreva("Olá, ", nome_usuario, "! Bem-vindo(a).\n") // Exibição direta
    }
}

Novos Tipos:

  • real: Armazena números que podem ter casas decimais, usando o ponto . como separador. 
    
programa {
    real preco_item, preco_com_acrescimo
    const real VALOR_ACRESCIMO = 5.50

    funcao inicio() {
        escreva("Digite o preço do item: R$")
        leia(preco_item)
        preco_com_acrescimo = preco_item + VALOR_ACRESCIMO
        escreva("Preço com acréscimo: R$", preco_com_acrescimo, "\n")
    }
}
  • caracter: Armazena um único caractere, delimitado por aspas simples ' '. 
    
programa {
    caracter inicial_nome

    funcao inicio() {
        escreva("Digite a primeira letra do seu nome: ")
        leia(inicial_nome)
        escreva("Sua inicial é: ", inicial_nome, "\n")
    }
}

Por que Diferentes Tipos?

Usar o tipo correto garante que o computador armazene e processe a informação eficientemente. Tipos numéricos (inteiro, real) permitem cálculos matemáticos. Tipos de texto (cadeia, caracter) permitem manipulação textual. Misturar tipos de forma inadequada (ex: guardar "abc" num inteiro) resulta em erro.

Exemplo Integrado (Contexto Tibia):

Vamos simular a leitura de alguns dados de um personagem no Tibia e exibir um resumo, utilizando os tipos que aprendemos.


programa {
    // Constantes e Variáveis Globais
    const cadeia NOME_JOGO = "Tibia"
    cadeia nome_char
    inteiro level_char
    real capacidade_char // Alterado de dinheiro_char para capacidade_char

    funcao inicio() {
         escreva("--- Cadastro Rápido de Personagem em ", NOME_JOGO, " ---\n")
         
         escreva("Nome do seu personagem: ")
         leia(nome_char)
         
         escreva("Level atual de ", nome_char, ": ")
         leia(level_char)
         
         escreva("Capacidade de carga (ex: 750.0 Oz): ")
         leia(capacidade_char)
         
         escreva("\n--- Resumo do Personagem ---\n")
         escreva("Nome: ", nome_char, "\n")
         escreva("Level: ", level_char, "\n")
         escreva("Capacidade: ", capacidade_char, " Oz\n")
    }
}

Este exemplo demonstra como diferentes tipos coexistem para representar dados variados. A prática com leia() e escreva() usando estes tipos é fundamental.
