Tabela de Consulta: Pseudo-classes

Use estas pseudo-classes para aplicar estilos especiais a elementos com base em seu estado ou posição.

Pseudo-classe O que faz? Exemplo de Uso
:hover Aplica um estilo quando o cursor do mouse está sobre um elemento. Essencial para interatividade. #barra-lateral a:hover { text-decoration: underline; color: #663322; }
(Sublinha e muda a cor do item de menu quando o mouse passa por cima).
:active Aplica um estilo no momento exato em que um elemento está sendo clicado. #barra-lateral a:active { color: red; }
(Muda a cor do link do menu para vermelho no exato instante do clique).
:visited Aplica um estilo a um link que já foi visitado pelo usuário. #barra-lateral a:visited { color: #8c6d5d; }
(Deixa os links de menu já visitados com uma cor de café mais clara).
:first-child Seleciona o primeiro elemento entre um grupo de irmãos. .item-cardapio:first-child { font-weight: bold; }
(Deixa o primeiro item do cardápio em negrito para destacá-lo).
:last-child Seleciona o último elemento entre um grupo de irmãos. .item-cardapio:last-child { margin-bottom: 0; }
(Remove a margem inferior do último item do cardápio para ajustar o espaçamento).
:focus Aplica um estilo a um elemento que recebeu foco (ex: link selecionado com a tecla Tab). Essencial para acessibilidade. a:focus { outline: 2px solid orange; }
(Adiciona uma borda de destaque em qualquer link focado via teclado).
:nth-child(n) Seleciona elementos com base em sua posição. Use even para pares e odd para ímpares. .item-cardapio:nth-child(even) { background-color: #fdfaf8; }
(Aplica um fundo suave nos itens pares do cardápio, criando um efeito "zebrado").

Tabela de Consulta: Pseudo-elementos

Use estes pseudo-elementos para estilizar partes específicas de um elemento ou para adicionar conteúdo decorativo.

Pseudo-elemento O que faz? Exemplo de Uso
::before Insere um conteúdo (texto ou símbolo) antes do conteúdo de um elemento. .item-cardapio::before { content: '☕ '; }
(Adiciona um ícone de xícara de café antes de cada item do cardápio).
::after Insere um conteúdo (texto ou símbolo) depois do conteúdo de um elemento. h1::after { content: ' ♥'; font-size: 0.8em; color: #ffb6c1; }
(Adiciona um coração decorativo depois do título principal).
::first-letter Seleciona e estiliza a primeira letra de um bloco de texto. #principal p::first-letter { font-size: 2em; color: #663322; float: left; margin-right: 5px;}
(Cria um efeito de letra capitular no parágrafo principal).
::first-line Seleciona e estiliza a primeira linha de um bloco de texto. #principal p::first-line { font-weight: bold; }
(Deixa toda a primeira linha do parágrafo principal em negrito).
::selection Aplica um estilo ao texto que é selecionado (destacado) pelo usuário. ::selection { background-color: #663322; color: white; }
(Muda a cor de fundo da seleção de texto para a cor de café, com texto branco).
::marker Permite estilizar o marcador (bullet point ou número) de um item de lista (<li>). .item-cardapio::marker { content: '☕'; font-size: 1.2em; }
(Substitui o marcador padrão do cardápio por um emoji de xícara de café).
::placeholder Permite estilizar o texto de placeholder em campos de formulário como <input>. input::placeholder { color: #aaa; font-style: italic; }
(Estiliza o texto de um futuro campo de busca, como "Pesquisar no cardápio...").
