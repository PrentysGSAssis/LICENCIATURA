Tabela de Tags de Agrupamento (Grouping Tags)

Tag Semântica Função Principal
<header> Define o cabeçalho da página ou de uma seção. Ideal para conter o logo, título principal e navegação.
<nav> Agrupa os links de navegação principais do site, como o menu principal.
<main> Delimita o conteúdo principal e único da página. Só deve haver uma por documento.
<section> Agrupa um bloco de conteúdos relacionados tematicamente (ex: "Sobre Nós", "Contato").
<article> Representa um conteúdo completo e independente que poderia ser distribuído por conta própria (ex: um post de blog, uma notícia).
<aside> Define um conteúdo complementar ou secundário, como uma barra lateral (sidebar) ou uma nota de rodapé contextual.
<footer> Define o rodapé da página ou de uma seção, contendo informações de autoria, direitos autorais, etc.
<div> É um contêiner genérico de bloco (block-level), não-semântico. Usado para agrupar elementos para fins de estilização quando nenhuma outra tag semântica se aplica.
<span> É um contêiner genérico em linha (inline-level), não-semântico. Usado para agrupar uma pequena parte de um texto ou conteúdo para aplicar estilos específicos.
