Parágrafos e Quebras de Linha

Você pode escrever um parágrafo inteiro aqui, de qualquer jeito. Basta colococar tudo no meio do par de tags <p> e </p>. Mas note que não importa quantas quebras de linha você insere no visual code. No fim das contas só entrará a quebra de linha se você usar a tag <br>

Claro que é possível realizar uma "quebra" de texto caso você crie um novo parágrafo, mas muitas vezes precisar criar uma quebra de texto durante o mesmo parágro, e não é nada prático ter que ficar criando parágrafos cada vez que você precisa criar uma nova linha.

Então, para fixar, esta tag, e este processo de ENTENDER que a organização do código HTML é extremamente importante, faremos um novo exercídio hoje. Selecione uma musica a sua escolha, e coloque a letra dela em seu site.

Atente-se, as quebras de linhas devem ser inseridas quando necessário. Nada e criar um novo parágrafo para cada verso, MAS cada Estrofe é sim um novo parágrafo!
