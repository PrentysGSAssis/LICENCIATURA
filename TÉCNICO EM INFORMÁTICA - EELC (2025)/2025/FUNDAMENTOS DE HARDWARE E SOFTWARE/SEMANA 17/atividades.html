Semana 17: A Instalação do Windows – Do Boot à Primeira Tela

Introdução: O Começo de Tudo

Olá, turma! Nesta semana, daremos um passo crucial na nossa jornada de domínio do Windows: vamos entender como se inicia o processo de instalação do sistema operacional. Instalar o Windows "do zero" (também chamado de instalação limpa) é uma habilidade fundamental para técnicos e usuários avançados, permitindo resolver problemas graves, otimizar o desempenho ou simplesmente começar com um sistema "fresco". Hoje, focaremos nas etapas que antecedem a cópia dos arquivos para o disco.

Atenção: Esta semana coincide com a aplicação de provas parciais. O conteúdo teórico é essencial para a compreensão, e a prática em máquinas virtuais será adaptada conforme o tempo disponível.

1. O Que Precisamos Antes de Começar? (Pré-requisitos)

Antes de pensar em formatar um computador e instalar o Windows, precisamos de alguns itens e informações:

  • Arquivo ISO do Windows: Um arquivo ISO é uma imagem de disco, como uma cópia digital exata de um DVD de instalação. Você pode baixar o arquivo ISO oficial do Windows diretamente do site da Microsoft. Ele contém todos os arquivos necessários para a instalação.
  • Licença do Windows (Chave do Produto - Product Key): Para usar o Windows legalmente, você precisa de uma licença. A chave do produto é um código de 25 caracteres que ativa sua cópia do Windows. Em muitos computadores de marca (notebooks, desktops), a chave já vem embutida no firmware (UEFI/BIOS) ou em um adesivo. Se estiver montando um PC ou reinstalando, precisará da chave que adquiriu. É possível instalar o Windows sem inserir a chave inicialmente e ativá-lo depois.
  • Requisitos Mínimos de Hardware: Cada versão do Windows tem requisitos mínimos de processador, memória RAM, espaço em disco e placa gráfica. É importante verificar se o computador atende a esses requisitos para garantir um bom funcionamento.
  • Mídia de Instalação (Pendrive): O método mais comum hoje é usar um pendrive USB. Precisaremos de um pendrive com capacidade suficiente (geralmente 8GB ou mais) que será formatado (apagado) durante o processo de criação da mídia.

2. Criando a Mídia de Instalação Bootável

Com o arquivo ISO em mãos, precisamos transferi-lo para o pendrive de uma forma especial que o torne "bootável", ou seja, que permita ao computador iniciar diretamente por ele.

  • Media Creation Tool (Ferramenta de Criação de Mídia da Microsoft): É a ferramenta oficial e mais recomendada pela Microsoft para baixar o Windows e criar um pendrive de instalação diretamente. Ela guia o usuário por todo o processo.
  • Outras Ferramentas (Ex: Rufus): Existem softwares de terceiros, como o Rufus, que são populares para criar mídias bootáveis a partir de arquivos ISO. Eles oferecem mais opções de personalização, mas exigem um pouco mais de atenção.

Lembre-se: Ao criar a mídia de instalação, o pendrive será formatado e todos os dados nele serão perdidos!

3. BIOS/UEFI: A Chave da Ignição do Computador

Antes que o Windows (ou qualquer sistema operacional) possa carregar, um software fundamental entra em ação assim que você liga o computador: o firmware da placa-mãe.

  • BIOS (Basic Input/Output System): É o firmware mais antigo, presente em computadores mais velhos. Possui uma interface baseada em texto, controlada pelo teclado.
  • UEFI (Unified Extensible Firmware Interface): É o sucessor moderno da BIOS, presente na maioria dos computadores atuais. Oferece uma interface gráfica mais amigável (muitas vezes com suporte a mouse), inicialização mais rápida e recursos de segurança aprimorados, como o "Secure Boot".

A principal função da BIOS/UEFI é realizar um auto-teste inicial do hardware (POST - Power-On Self-Test), verificar os dispositivos conectados e, em seguida, carregar o sistema operacional a partir de uma unidade de armazenamento definida.

Como Acessar a BIOS/UEFI?

Para acessar a tela de configuração da BIOS/UEFI, você precisa pressionar uma tecla específica logo após ligar o computador, antes que o logo do Windows apareça. As teclas mais comuns são:

  • DEL (Delete)
  • F2
  • F1
  • F10
  • F12
  • ESC (Escape)

A tecla correta varia conforme o fabricante da placa-mãe ou do computador. Geralmente, uma mensagem rápida aparece na tela indicando qual tecla pressionar (ex: "Press DEL to enter Setup").

4. Definindo a Ordem de Boot (Prioridade de Inicialização)

Uma vez dentro da BIOS/UEFI, uma das configurações mais importantes para a instalação do Windows é a ordem de boot (ou "Boot Priority", "Boot Sequence"). Essa configuração diz ao computador qual dispositivo ele deve tentar iniciar primeiro, depois o segundo, e assim por diante.

Para instalar o Windows a partir do nosso pendrive, precisamos garantir que o "Dispositivo USB" ou "Removable Device" esteja listado como a primeira opção na ordem de boot, antes do disco rígido (HDD/SSD) interno do computador.

Essa opção geralmente se encontra em abas chamadas "Boot", "Startup" ou "Advanced BIOS Features". Após alterar a ordem, é crucial salvar as alterações e sair da configuração da BIOS/UEFI (geralmente com a tecla F10 ou uma opção no menu "Exit"). O computador irá reiniciar.

5. As Primeiras Telas da Instalação do Windows

Se a ordem de boot estiver correta e o pendrive de instalação estiver conectado, após o reinício, você deverá ver uma mensagem como "Pressione qualquer tecla para iniciar do CD ou DVD..." (mesmo que seja um USB) ou o instalador do Windows carregará diretamente.

[Imagem de exemplo: Tela inicial do instalador do Windows com seleção de idioma]
  1. Idioma, Formato de Hora e Teclado: A primeira tela interativa pedirá para você selecionar o "Idioma a instalar", "Formato de hora e moeda" e "Teclado ou método de entrada". Para nós no Brasil, geralmente será:
    • Idioma a instalar: Português (Brasil)
    • Formato de hora e moeda: Português (Brasil – ABNT)
    • Teclado ou método de entrada: Português (Brasil – ABNT2) - verifique o layout do seu teclado.
    Clique em "Avançar".
  2. Instalar Agora: Na próxima tela, clique no botão grande "Instalar agora".
  3. Chave do Produto (Product Key): Pode surgir uma tela pedindo a chave do produto do Windows.
    • Se você tiver a chave, pode digitá-la.
    • Se não tiver no momento, ou se a chave estiver embutida no firmware (comum em notebooks), você pode clicar em "Não tenho a chave do produto". A instalação continuará, e você poderá ativar o Windows depois.
  4. Seleção da Edição do Windows: Se o seu arquivo ISO ou mídia de instalação contiver múltiplas edições do Windows (como Home, Pro, Education), uma tela aparecerá para você escolher qual edição deseja instalar. Escolha a que corresponde à sua licença.
  5. Termos de Licença: Leia (ou finja que leu 😉) os termos de licença do software. Marque a caixa "Aceito os termos da licença" e clique em "Avançar".
  6. Tipo de Instalação: Esta é uma tela crucial! Você verá duas opções:
    • Atualização: Instalar o Windows e manter arquivos, configurações e aplicativos. Esta opção só está disponível se você estiver executando o instalador a partir de uma versão anterior do Windows já funcional e serve para atualizar o sistema.
    • Personalizada: Instalar apenas o Windows (avançado). Esta é a opção que geralmente escolhemos para uma instalação limpa. Ela permite que você gerencie as partições do disco rígido, formate-as e escolha exatamente onde o Windows será instalado. É sobre esta opção que focaremos na próxima etapa do nosso aprendizado.
    [Imagem de exemplo: Tela de "Que tipo de instalação você deseja?"]
    Selecione "Personalizada".
