Tenho um outro plano para este conteúdo, iremos trabalhar um projeto:PROJETO - Desenvolvimento de LógicaEntendendo que compreender Lógica de Programação e Algoritmos é parte FUNDAMENTAL da formação de um técnico em Informática e visto que os alunos tiveram muita dificuldade para compreender o conteúdo do Módulo I, foi idealizado por mim um projeto de trabalhar com os alunos o projeto de Portugol Studio. Sendo que utilizarei 2 aulas para o projeto de Portugol e 2 aulas para Sistemas Operacionais.Minha ideia é o projeto durar durante todo o semestre, avançando lentamente mas progressivamente, utilizando 2 ou até 3 semanas para trabalhar cada um dos os conceitos teóricos mais básicos e fundamentais, como a ideia de que qd estamos programando estamos na verdade "alocando" memórias e dando comandos ao processador, incluir analogias pontuais com explicações, como: pegamos um problemas, pensamos em um algoritmo para resolver este problema, e só então "traduzimos" este algoritmo para uma lingua que o computador entenda. E que programar em Python ou Java a partir de um algoritmo seria algo similar a pegar um texto e traduzilo para portugues ou ingles.
